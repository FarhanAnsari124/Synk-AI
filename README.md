# Synk-AI 🚀

A production-ready full-stack Gen AI job preparation platform.

## Features
- User authentication (JWT)
- Resume upload & analysis
- AI-powered interview questions
- Skill gap detection

## Tech Stack
- Node.js
- Express.js
- MongoDB

## Setup

```bash
git clone https://github.com/FarhanAnsari124/Synk-AI.git
cd Synk-AI
npm install
npm run dev
